FreeBot
Experimental Telegram bot con AI uncensored para research.

## Env Vars
- TELEGRAM_TOKEN: Tu bot token.
- MODEL_API_KEY: NVIDIA API key.
- OWNER_ID: Tu user ID.
- DB_FILE: wormgpt_v3.sqlite3 (default).

## Deploy
1. Clona: git clone https://github.com/retro935/WormGPT.git
2. pip install -r requirements.txt
3. python app.py

## Comandos
- /start: Bienvenida VIP con video.
- /addvip <id> <days>: Admin add.
- Mensajes: AI responde con DeepSeek.
